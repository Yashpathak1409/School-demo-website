## ✨ Features & Modules

The application features 12 comprehensive pages/sections matching administrative and educational requirements:

1.  **🏠 Home Page**: 
    *   Responsive hero banner showcasing the **actual school building facade**.
    *   Urgent marquee notice board for announcements (admissions, test dates, sports meet).
    *   Student statistics card block (Enrolled Students, Teachers, CBSE success rate).
    *   Welcoming message from Principal Mr. Devendra Singh with a signature block.
2.  **ℹ️ About Us**: School history, vision & mission, core values, and message board from the Managing Director.
3.  **📚 Academics & Courses**: 
    *   Syllabus breakdown for Primary, Secondary, Senior Secondary wings (Science, Commerce, Arts).
    *   **Sports & Athletics Wing**: Detailing team sports, yoga pavilions, physical fitness testing frameworks, and schedules.
    *   Interactive detail modals displaying timings, lab infrastructure, and assessment plans.
4.  **🖼️ School Gallery**: 
    *   Interactive grid filters (Labs, Sports, Cultural, Campus).
    *   Loaded with **actual school photographs** representing award functions, morning prayers, and staff meetings.
    *   Click-to-zoom fullscreen Image Lightbox overlay.
5.  **📝 Online Result Checking**: 
    *   Roll number search input (Demo Roll Numbers: `202601`, `202602`, `202603`).
    *   **Board Toppers Roll of Honor**: Banners of Class X and XII toppers (Aditya Kumar Singh 97%, Mayank Sharma 93%) with zoom popups.
6.  **🎓 Result Detail Marksheet**: Digital scorecard sheet displaying subject marks, grand totals, grade computations, overall percentage, and a printable layout view.
7.  **📄 Transfer Certificate (TC) Portal**: 
    *   Online TC verification system searchable by certificate ID (Demo IDs: `TC-202601`, `TC-202602`).
    *   Displays verified parchment-styled certificate files.
    *   Digital application request form for students/parents requesting a TC.
8.  **👨‍🏫 Faculty Team**: Team profile grids listing qualifications, qualifications, and mail links.
9.  **📰 News & Blogs**: Categorized articles on school activities, boards counseling study tips, and exhibitions.
10. **🏥 Hostel Residence (Hostal)**: Residential wing descriptions, weekly mess dining schedules (Breakfast, Lunch, Dinner), and boarding request panels.
11. **📁 Mandatory Disclosures**: Compliance documents table (Society registration, Safety NOCs, fire certificates) with download simulations.
12. **📞 Contacts & FAQs**: Direct calling phone cards, map coordinates links, inquiry form, and collapsible FAQ accordion.

## 🛠️ Project Run Instructions

### Prerequisites
*   [Node.js](https://nodejs.org/) (Version >= 18.0)
*   [npm](https://www.npmjs.com/)

### 1. Install Dependencies
Navigate to the root workspace directory and run:
```bash
npm install
```

### 2. Launch Development Server
Start the local server with hot module replacement (HMR):
```bash
npm run dev
```
Open your browser and navigate to `http://localhost:5173`.

### 3. Production Build
To build the optimized client-side bundle under `dist`:
```bash
npm run build
```
To preview the production bundle locally:
```bash
npm run preview
```
